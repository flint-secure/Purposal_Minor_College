# Manual download guide

Some publishers block automated downloads. Use your **browser** (or NCIT library / IEEE Xplore / ScienceDirect access) and save each file into `references/papers/` using the exact filename below.

## Step-by-step

### 03 — Laperdrix et al. (2016)

1. Open https://hal.inria.fr/hal-01285470/file/beauty-sp16.pdf  
2. Save as: `03-laperdrix-2016-beauty-and-beast.pdf`

### 04 — Vastel et al. (2018)

1. Open https://inria.hal.science/hal-01652021  
2. Click **Download** / open the PDF attachment  
3. Save as: `04-vastel-2018-fp-stalker.pdf`  
   (Or use IEEE Xplore with college login: DOI `10.1109/SP.2018.00008`)

### 05 — Bolton & Hand (2002)

1. Open https://projecteuclid.org/journals/statistical-science/volume-17/issue-3/Statistical-Fraud-Detection-A-Review/10.1214/ss/1042727940.pdf  
2. Save as: `05-bolton-hand-2002-fraud-review.pdf`  
   (This article is usually **free** on Project Euclid when downloaded in a browser.)

### 06 — West & Bhattacharya (2016)

1. Open https://doi.org/10.1016/j.cose.2015.09.005  
2. Use **ScienceDirect** through Pokhara University / NCIT library if asked to pay  
3. Save as: `06-west-bhattacharya-2016-fraud-review.pdf`  
4. Delete any small broken file left from a failed automatic download (~1 MB)

### 07 — Fiore et al. (2019)

1. Open https://doi.org/10.1016/j.ins.2019.045 or search *Information Sciences* 479 (2019)  
2. Download PDF via institutional access  
3. Save as: `07-fiore-2019-gan-fraud.pdf`

### 09 — Mjaaland et al. (2021)

1. Open https://doi.org/10.1109/CNS48642.2021.9470941  
2. Download through **IEEE Xplore** (college subscription)  
3. Save as: `09-mjaaland-2021-sim-swap.pdf`
