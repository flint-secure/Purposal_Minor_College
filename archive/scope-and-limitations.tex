\section{Scope and Limitations}

\subsection{Scope}
The project will include a Go REST API with device identification and transaction scoring endpoints, a web SDK,
two Redis instances, NATS workers, PostgreSQL storage, rule-based scoring for Nepal, system design diagrams,
and automated plus load testing.

\subsection{Limitations}
\begin{itemize}
  \item The first version will use clear weighted rules, not machine learning.
  \item Recipient risk will be weak until enough fraud labels are collected.
  \item Location lookup (GeoIP) may be less accurate for some Nepali networks.
  \item Official FIU blocklists will be updated manually until an automated feed exists.
\end{itemize}
